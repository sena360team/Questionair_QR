```

│  🛠️ Form Builder                                    [💾 บันทึก] │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│  │ 🔽 Dropdown      │              │  แบบสอบถามความพึงพอใจ  │ │
│  │ ⭕ ตัวเลือกเดียว │              │                         │ │
│  │ ☑️ หลายตัวเลือก │              │  ┌─ 👤 ข้อมูล ───────┐  │ │
│  │ ⭐ ให้คะแนน      │              │  │ ชื่อ: [________] │  │ │

│  └──────────────────┘              │                         │ │
│                                    └─────────────────────────┘ │

│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ⋮⋮  📦 Section: ข้อมูลส่วนตัว    [✏️] [🗑️] [↑] [↓]    │   │
│  └─────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ⋮⋮  📝 ชื่อ-นามสกุล *            [✏️] [🗑️] [↑] [↓]    │   │

│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ⋮⋮  🔽 จังหวัด *                [✏️] [🗑️] [↑] [↓]    │   │
│  │     ตัวเลือก: กรุงเทพฯ, เชียงใหม่...                    │   │
│  └─────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ⋮⋮  📦 Section: ข้อมูลบริการ    [✏️] [🗑️] [↑] [↓]    │   │
│  └─────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ⋮⋮  🔽 ประเภทบริการ              [✏️] [🗑️] [↑] [↓]    │   │
│  │     อื่นๆ (ระบุเอง): ✅                                   │   │
│  └─────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ⋮⋮  ⭐ ความพึงพอใจ              [✏️] [🗑️] [↑] [↓]    │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  [+ เพิ่มคำถามใหม่]                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**Field Types ที่ใช้ได้:**
| Icon | Type | ใช้สำหรับ |
|------|------|-----------|
| 📦 | Section | แบ่งกลุ่มคำถาม (Card) |
| 📝 | ข้อความสั้น | Text input |
| 📄 | ข้อความยาว | Textarea |
| 🔽 | Dropdown | เลือกจากรายการ |
| ⭕ | ตัวเลือกเดียว | Radio buttons |
| ☑️ | หลายตัวเลือก | Checkboxes |
| ⭐ | ให้คะแนน | Rating 1-5 ดาว |
| 📅 | วันที่ | Date picker |
| 📱 | เบอร์โทร | Phone (จับเฉพาะตัวเลข) |

**Actions:**
- กดเพิ่ม field
- แก้ไข field (คลิก ✏️)
- ลากเรียงลำดับ (drag ⋮⋮)
- ดูตัวอย่าง (preview panel)
